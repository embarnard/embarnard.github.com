About

PianWHOA! is a virtual piano with a tutor mode programmed in C++ using QT Creator. This project was created by Emilie Menard Barnard(@embarnard), Brittany Berin(@bmbritt10), Bernice Chen, and Kristen Morse for CS 48 at UC Santa Barbara in Winter 2011.

Download Instructions

Download the .tar or .zip file and save to your computer. Consult the QT downloads page and download the appropriate version of the QT library (this is necessary for PianWHOA! to run). Then open the Piano1.app file to start playing!
