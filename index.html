GitHub Projects

Cellular Automata Encryption

I've created a cellular automata encryption generator that allows users to learn about CA Encryption techniques in a visual manner. The python and javascript code can be found here. It allows the user to input any CA rule (mod 256) and set output preferences. If you want to see the generator in action, click here.

PianWHOA!

PianWHOA! is a virtual learning piano programmed in C++ using QT Creator. I collaborated with Brittany Berin, Bernice Chen and Kristen Morse on this project. It allows users to play a virtual piano in either Free Play or Tutor Mode. Tutor Mode features five pre-loaded songs that prompts the user to press the keys as they light up to play the tune.

Awesome Library

I'm currently working with Professor Phill Conrad on the Awesome library. Why "Awesome?" It's a Discrete Math library, but it's the exact opposite of DisMaL. I'm primarily working with javascript to automate question generation. This will allow students to self-study various problems related to Discrete Math, such as set operations and relation classification.
